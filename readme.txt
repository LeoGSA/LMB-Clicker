RUSSIAN

Эта программа - автокликер.
Она полезна для некоторых браузерных игр, особенно для тех, где нужно много раз кликать в одну точку (вырубка ресурсов, собирание подарков и т.д.).
Вы наводите мышку в нужное место и нажимаете Пробел. Программа делает 50 кликов в то место, куда была наведена мышка.

Программу можнозапускать из командной строки. В этом случае вы можете указать, сколько кликов должна делать программа. Число указывается первым аргументом.

-----------------------------------------------

Например:

python mouse-test.py
Это просто запуск программы по умолчанию. Одно нажатие пробела - 50 кликов.


python mouse-test.py 30
Запуск программы с указанием желательного количества кликов. В данном случае - 30 кликов.
